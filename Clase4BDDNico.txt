Para las claves es siempre el mismo tipo de dato en las tablas.

Situación 1:
- Un gimnasio anotaba todo el excel. 

Clientes: 
ID (PK) - DNI - Apellido - Nombre - Celular - Calle - Altura

Profesores:
ID (PK) - DNI (UX) - Apellido - Nombre - Celular - Calle - Altura - CUIL (UX) - Legajo (UX)

Actividades: 
ID (PK) - ID Profesor (FK) - HoarioInicio - HorarioFin - Dia

Clases:
ID_Clase (FK) - ID_Actividad (FK) - ID_Cliente (FK) - Fecha - HoraIngreso - HoraSalida

Situación 2:
- Una clinica médica necesita organizar su agenda. 

Pacientes: 
ID (PK) - DNI - Apellido - Nombre - Celular - Calle - Altura

Médicos:
ID (PK) - DNI (UX) - Apellido - Nombre - Celular - Calle - Altura - CUIL (UX) - Legajo (UX) - Especialidad

Consulta: 
ID (PK) - ID Profesor (FK) - HoarioInicio - HorarioFin - Dia

Turnos:
ID_Turnos (FK) - ID_Paciente (FK) - ID_Medico (FK) - Fecha - HoraIngreso - HoraSalida
